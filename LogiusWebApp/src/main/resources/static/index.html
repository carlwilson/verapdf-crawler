This web application gives rough statistics on the document formats published on a given website. It uses Bing search to take a sample of up to 1000 documents on the domain you enter and counts the number of PDF, ODF, OOXML and Microsoft Ofice files found. It also gives detailed information about the PDF versions and standards found. To get more complete samples and more accurate statistics, please register. It's free but we reserve the right to prevent abuse of this tool.
